git同时推送2个仓库

发表于 2018-07-16 | 分类于 git |

怎么样将一个本地git仓库推送到2个不同的远程仓库呢?

方法一

  1. 本地仓库添加到远程网站,正常来即可
  2. 使用命令git remote add github https://github.com/xxxx添加新的仓库地址
  3. 推送第一个仓库git push origin master:master
  4. 推送第二个仓库git push github master:master

方法二

  1. 添加第二个仓库git remote set-url --add oginin https://github.com/xxxx
  2. 推送修改到所有仓库git push origin master:master
